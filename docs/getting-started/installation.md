# Installation

## Requirements

- Python 3.9+ (recommended)
- A virtual environment (e.g. `venv`, `conda`)

## Install from source

```bash
git clone https://github.com/ORG/opticstream.git
cd opticstream
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
