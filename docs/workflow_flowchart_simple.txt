OCT Pipeline Workflow Flowchart (Text Version)
===============================================

BATCH LEVEL PROCESSING
------------------------

[Batch Detected]
    |
    v
[Emit: linc.opticstream.psoct.batch.ready]
    |
    v
[process_tile_batch_flow]
    |
    +---> [Archive Tiles] (parallel)
    |         |
    |         v
    |    [Emit: linc.opticstream.psoct.batch.archived]
    |         |
    |         +---> [upload_to_linc_batch_flow]
    |         |         |
    |         |         v
    |         |    [Emit: linc.opticstream.psoct.batch.uploaded]
    |         |
    |         +---> [State Management Flow]
    |
    +---> [Convert Task] (parallel)
            |
            +---> [SPECTRAL] -> [Spectral to Complex]
            |                      |
            |                      v
            |                 [Emit: linc.opticstream.psoct.batch.complexed]
            |
            +---> [COMPLEX] -> [Link Complex Files]
                                 |
                                 v
                            [Emit: linc.opticstream.psoct.batch.complexed]
                                 |
                                 v
                            [complex_to_processed_batch_flow]
                                 |
                                 v
                            [Emit: linc.opticstream.psoct.batch.processed]
                                 |
                                 v
                            [State Management Flow]

STATE MANAGEMENT (Batch Level)
-------------------------------

[State Management Flow] (triggered by linc.opticstream.psoct.batch.processed or linc.opticstream.psoct.batch.archived)
    |
    v
[Check Batch State]
    |
    v
[Update Mosaic Artifact]
    |
    v
{All Batches Processed?}
    |
    +---> [NO] -> [End]
    |
    +---> [YES]
            |
            v
        {Already Stitched?}
            |
            +---> [YES] -> [Skip Event] -> [End]
            |
            +---> [NO]
                    |
                    v
                [Emit: linc.opticstream.psoct.mosaic.ready]

MOSAIC LEVEL PROCESSING
------------------------

[Emit: linc.opticstream.psoct.mosaic.ready]
    |
    v
[process_mosaic_flow]
    |
    v
{First Slice? (mosaic_id <= 2)}
    |
    +---> [YES]
    |       |
    |       v
    |   [Process Coordinates]
    |   - Fiji Stitch
    |   - Process Tile Coordinates
    |   - Generate Template
    |       |
    |       v
    |   [Template Ready]
    |
    +---> [NO]
            |
            v
        [Use Existing Template]
            |
            v
        [Template Ready]
            |
            v
        [Stitch AIP & MIP] (parallel)
            |
            v
        [Generate Mask from MIP]
            |
            v
        [Stitch Enface Modalities] (parallel)
        - ret, ori, biref, surf
            |
            v
        [Emit: linc.opticstream.psoct.mosaic.enface_uploaded]
            |
            +---> [State Management Flow]
            |
            +---> [Slack Notification Flow]

STATE MANAGEMENT (Slice Level)
-------------------------------

[State Management Flow] (triggered by linc.opticstream.psoct.mosaic.enface_uploaded)
    |
    v
[Check Slice State]
    |
    v
[Update Slice Artifact]
    |
    v
{Both Mosaics Stitched?}
    |
    +---> [NO] -> [End]
    |
    +---> [YES]
            |
            v
        [Emit: linc.opticstream.psoct.slice.ready]

SLICE LEVEL PROCESSING
-----------------------

[Emit: linc.opticstream.psoct.slice.ready]
    |
    v
[register_slice_flow]
    |
    v
[Thruplane Registration]
- Align normal and tilted illuminations
- Generate 3D axis vectors
    |
    v
[Emit: linc.opticstream.psoct.slice.registered]

VOLUME PROCESSING (Optional)
-----------------------------

[process_mosaic_flow]
    |
    v
{Stitch 3D Volumes?}
    |
    +---> [NO] -> [Skip]
    |
    +---> [YES]
            |
            v
        [Stitch Volume Modalities] (parallel)
        - dBI, R3D, O3D
            |
            v
        [Emit: linc.opticstream.psoct.mosaic.volume_stitched]

CONDITIONS SUMMARY
==================

Batch Level:
- All Batches Processed: processed_batches == total_batches
- Already Stitched: flag file exists OR AIP file exists

Mosaic Level:
- First Slice: mosaic_id <= 2 OR force_refresh_coords == True
- Stitch 3D Volumes: stitch_3d_volumes == True AND volume_modalities not empty

Slice Level:
- Both Mosaics Stitched: 
  * Normal mosaic stitched (check_mosaic_stitched_task)
  * Tilted mosaic stitched (check_mosaic_stitched_task)

EVENT CATALOG
=============

Batch Events:
- linc.opticstream.psoct.batch.ready: Batch detected
- linc.opticstream.psoct.batch.complexed: Complex data ready
- linc.opticstream.psoct.batch.processed: Batch processing complete
- linc.opticstream.psoct.batch.archived: Tiles archived
- linc.opticstream.psoct.batch.uploaded: Batch uploaded to LINC

Mosaic Events:
- linc.opticstream.psoct.mosaic.ready: All batches processed
- linc.opticstream.psoct.mosaic.enface_uploaded: Enface outputs uploaded
- linc.opticstream.psoct.mosaic.volume_stitched: Volume modalities stitched
- linc.opticstream.psoct.mosaic.volume_uploaded: Volumes uploaded

Slice Events:
- linc.opticstream.psoct.slice.ready: Both mosaics stitched
- linc.opticstream.psoct.slice.registered: Slice registration complete
